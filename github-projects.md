### 一句话介绍
> 命令行下的模糊搜索神器，配合 shell 用起飞

### 为什么有趣 ✨
- 启动毫秒级，体感丝滑
- 可作为 vim / tmux / git 的增强器
- 完全可定制，配置文件极简

### 适用场景
- 快速查找历史命令 / 文件
- git checkout 分支时模糊匹配

### 快速上手
```bash
brew install fzf
$(brew --prefix)/opt/fzf/install
```

### 踩坑记录 🐛
- Windows 下需要用 scoop 或 wsl，体验略差

### 我的评价
⭐⭐⭐⭐⭐ 一旦用上就回不去的工具

### 相关项目
- [skim](https://github.com/lotabout/skim) — Rust 实现的 fzf 替代品
- [fzf-git.sh](https://github.com/junegunn/fzf-git.sh) — 官方 git 集成脚本
